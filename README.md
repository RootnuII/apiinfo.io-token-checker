# APIinfo.io Token Checker

This Node.js script is designed to generate and Bruteforce API tokens using the APIinfo.io API. It leverages the Cluster module to create multiple worker processes for parallel token generation and checking.

## Requirements

- Node.js
- npm (Node Package Manager)
